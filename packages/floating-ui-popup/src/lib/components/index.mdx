# Popup

A component that can be used to display additional information or actions. From a technical perspective, it is using [floating-ui](https://floating-ui.com/) as a base. Includes normal components likes modals, popovers, tooltips. Also includes more complex components like a popconfirm, dialog, etc.

## Basic usage

```tsx
<Popup trigger={<button>Click me</button>}>
  <div>Popup content</div>
</Popup>
```
